# Immersion_HTML

Ce README fournit la correction d'un exercice portant sur le développement HTML et CSS. L'exercice consistait à créer une page web selon certaines spécifications données, en utilisant HTML pour la structure et CSS pour la mise en forme.

voici la construction des corrections proposer :

- Immersion HTML 01 => notion de base
- Immersion HTML 02 => manipulation des balises plus avancés
- Immersion HTML 03 => manipulation des images
- Immersion HTML 04 => responsive
