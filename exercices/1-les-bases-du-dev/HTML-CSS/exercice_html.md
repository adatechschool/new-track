# Exercice HTML

## 1. Mon premier code

Vous devez écrire un titre `Salut, c'est mon code`

## 2. Mon premier code 2 : le retour

Vous devez rajouter un paragraphe `j'écris ce code sur mon ordi`

## 3. Petite présentation

maintenant écrivez votre nom / prénom en titre et une description de vous en paragraphe

(Vous pouvez regarder la deuxième partie)

## 4. organisons le tout

Nous allons changer les titre utiliser en de plus petit titre,
nous voudrons donc : \
Nom / prenom en titre
age en sous-titre
ville en sous-tire
description / introduction de notre personne en paragraphe

## 5. suite de l'organisation

Utiliser une balise pour ranger le contenu de vos informations

(Vous pouvez passer à la troisième partie)

## 6. toujours plus d'information ...

Maintenant il faudrait que tu rajoutes une image de vous

## 7. Exercice 7

Mettons en place un lien cliquable sur un réseau au choix

## 8. Formulaire simple

Créez un petit formulaire avec :

- un champ `nom` (type text)
- un champ `email` (type email)
- un bouton `Envoyer`

Écrire le formulaire dans une balise `<form>`.

## 9. Liste de compétences

Faites une liste (ordonnée ou non) avec 5 compétences que vous avez.  
Utilisez `<ul>` ou `<ol>` et `<li>`.

## 10. Table de données

Créez une petite table avec 3 colonnes : `Matière`, `Note`, `Commentaire` et 3 lignes de données.

## 11. Utiliser des classes et des id

Refaites votre présentation (nom, âge, ville, description) et ajoutez :

- une `class="profil"` sur le conteneur
- un `id="nom-prenom"` sur le titre du nom

## 12. Image responsive

Ajoutez une image avec un attribut `alt` et rendez-la responsive en utilisant l'attribut `style` ou une classe (`max-width: 100%; height: auto;`).

## 13. Lien vers une partie de la page

Ajoutez un lien en haut de la page qui, lorsqu'on clique dessus, fait défiler jusqu'à la section `description` (utilisez un `id` et un `a href="#id"`).

## 14. Média (audio / vidéo)

Insérez une balise `<audio>` ou `<video>` avec un fichier libre (ou un lien) et ajoutez les contrôles (`controls`) pour pouvoir lire/mettre en pause.
