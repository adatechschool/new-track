## Exercice 1

Afficher un message de présentation avec votre prénom

exemple attendu :

```bash
Hello, je m'appelle Charles
```

## Exercice 2

Afficher un message de présentation avec votre prénom \
votre prénom devra être contenu dans une déclaration de variable

exemple attendu :

```bash
Hello, je m'appelle Charles
```

## Exercice 3

Afficher un message de présentation avec votre prénom et votre age

exemple attendu :

```bash
Hello, je m'appelle Charles j'ai 30 ans
```

## Exercice 4

Afficher deux message de présentation

exemple attendu :

```bash
Hello, je m'appelle Charles, j'ai 30 ans
Hello, je m'appelle Marie, j'ai 30 ans
```

## Exercice 5

Même demande que **l'exercice 4** mais dans une seule déclaration de variable

exemple attendu :

```bash
Hello, je m'appelle Charles, j'ai 30 ans
Hello, je m'appelle Marie, j'ai 30 ans
```

## Exercice 6

Afficher la somme des deux ages

```bash
Charles à 30 ans et Marie aussi ce qui fait 60 ans pour les deux
```
