# Git en groupe

Pour l'instant tu vas vu 2 des principaux interêts de Git :

-   garder un historique de ton travail grâce aux commits
-   avoir une version sauvegardée en ligne de ton travail grâce à `git push` et à Github

Mais le principal interet de Git est le travail de groupe.

Dans ce guide, on va voir ce qu'il faut connaître afin de travailler correctement en groupe en utilisant Git et Github.

Si tu as le moindre doute sur les notions de bases, tu peux aller voir le git sur [les bases de Git](./les-bases-de-git.md).
