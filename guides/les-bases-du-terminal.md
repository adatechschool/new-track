# Les commandes Terminal

## Les indispensables

- **cd** : se déplacer dans l'arborescence (changer de dossier).
- **ls** : afficher la liste des fichiers et dossiers du répertoire courant.
- **mkdir** : créer un nouveau dossier.
- **touch** : créer un fichier vide ou mettre à jour sa date de modification.
- **pwd** : afficher le chemin absolu du répertoire courant.
- **man** : afficher la page de manuel d'une commande (expliquer son usage).
- **rm -r** : supprimer récursivement un dossier et tout son contenu.

## Les bonus

- **mv** : déplacer ou renommer un fichier ou dossier.
- **cp** : copier un fichier ou dossier.

Les commandes du terminal sont des instructions tapées pour effectuer rapidement des opérations sur le système de fichiers et les programmes. Parmi les commandes courantes utiles : cd, ls, mkdir, chmod, pwd, mv, rm, man, grep, echo. Utilisez l'autocomplétion (Tab) et l'historique des commandes (flèches haut/bas) pour gagner du temps et améliorer votre productivité.

## Autres commandes utiles

- **chmod** : changer les permissions d'un fichier ou dossier.
- **chown** : changer le propriétaire d'un fichier ou dossier.
- **sudo** : exécuter une commande avec les privilèges administrateur.
- **find** : rechercher des fichiers selon différents critères.
- **grep** : rechercher du texte dans les fichiers (déjà cité, très utile).
- **cat** : afficher le contenu d'un fichier.
- **less / more** : parcourir le contenu d'un fichier page par page.
- **head / tail** : afficher le début / la fin d'un fichier (tail -f pour suivre en continu).
- **ps** : lister les processus en cours.
- **top / htop** : surveiller l'utilisation des ressources et processus (htop à installer).
- **kill / killall** : terminer un processus par PID ou par nom.
- **df** : afficher l'espace disque disponible.
- **du** : afficher la taille des dossiers et fichiers.
- **tar** : compresser / décompresser des archives (tar -czf, tar -xzf).
- **zip / unzip** : compresser et décompresser des fichiers zip.
- **curl / wget** : télécharger des fichiers ou interagir avec des APIs.
- **ssh** : se connecter à distance à une machine.
- **scp / rsync** : copier des fichiers entre machines (rsync pour synchronisation).
- **git** : gestion de versions (commit, push, pull, branch...).
- **which / whereis** : localiser une commande binaire.
- **history** : afficher l'historique des commandes.
- **open** (mac) : ouvrir un fichier ou dossier avec l'application par défaut.
- **pbcopy / pbpaste** (mac) : copier dans / coller depuis le presse-papier système.

Utilisez ces commandes avec **précaution** `sudo`, `rm -r`, `chmod`, `chown`. Entraînez-vous dans un **répertoire** de test avant d'opérer sur **des données importantes**.
